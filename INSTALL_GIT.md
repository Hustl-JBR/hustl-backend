# Install Git

You need Git installed to push code to GitHub.

## Option 1: Install Git (Recommended)

1. **Go to:** https://git-scm.com/download/win
2. **Download** the Windows installer
3. **Run the installer:**
   - Click "Next" through all prompts
   - **IMPORTANT:** Make sure "Git from the command line" is selected
   - Click "Install"
4. **Restart Command Prompt** (close and reopen)
5. **Verify:**
   ```cmd
   git --version
   ```
   You should see a version number

## Option 2: Use GitHub Desktop (Easier!)

GitHub Desktop includes Git and is easier to use:

1. **Go to:** https://desktop.github.com
2. **Download** GitHub Desktop
3. **Install** it
4. **Sign in** with your `Hustl-JBR` account
5. **Add your repository:**
   - File → Add Local Repository
   - Browse to: `C:\Users\jbrea\OneDrive\Desktop\hustl-backend`
   - Click "Add repository"
6. **Publish to GitHub:**
   - Click "Publish repository" button
   - Make sure it says `Hustl-JBR/Hustl`
   - Click "Publish"

## After Installing

Once Git is installed, you can use the commands from GitHub, or use GitHub Desktop (much easier!).

## Quick Test

After installing, open Command Prompt and run:
```cmd
git --version
```

If you see a version number, Git is installed! ✅
